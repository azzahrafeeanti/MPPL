Lokasi – Event Offline

Kamu dapat melampirkan rincian lokasi event offline diselenggarakan, pastikan alamat sesuai dengan event yang akan kamu buat ya!

Event Online
maksimal 50 karakter
  • Silakan masukkan URL Streaming dengan benar karena URL ini akan diterima oleh pembeli tiket kamu setelah transaksi selesai dibayar.
  • Kamu tidak dapat melakukan perubahan URL Streaming setelah kamu selesai melakukan pembuatan event.
  • Jika kamu sudah membuat event online, kamu tidak bisa merubah menjadi event offline.
